# Cursor POC

As part of technical assessment I have performed the POC to connect cursor with github and pushing [readme](http://readme.md) file to the repository.

## Tools Installed

1. Cursor IDE
2. Claude code extension
3. Codex extension
4. Git

## What I Completed

1. Installed Cursor IDE.
2. Added and logged into the Claude Code extension.
3. Created a Public Github repository.
4. Opened the repository in cursor.
5. Created this README file.
6. Commited and pushed the changes to Github.
